# Projektidee/Projektbeschreibung/Projektaufgaben

Der BudgetPlanner ist eine Anwendung zur Verwaltung persönlicher Finanzen.

# Ziele:

Ziel der Anwendung ist es, dem Benutzer zu helfen, seine Einnahmen und
Ausgaben zu verwalten und einen Überblick über sein monatliches Budget
zu erhalten.

# Geplante Funktionen:

- Einnahmen hinzufügen
- Ausgaben hinzufügen
- Kategorien für Ausgaben erstellen
- Monatsübersicht
- Übersicht über Einnahmen und Ausgaben
- Budgetübersicht pro Monat
