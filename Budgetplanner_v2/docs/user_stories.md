# User Stories

## Muss-Funktionen

1. Als Benutzer moechte ich Konten erfassen, damit ich Bankkonto, Bargeld und Sparkonto getrennt betrachten kann.
2. Als Benutzer moechte ich Kategorien erfassen, damit Einnahmen und Ausgaben geordnet werden.
3. Als Benutzer moechte ich Einnahmen erfassen, damit mein verfuegbares Geld sichtbar wird.
4. Als Benutzer moechte ich Ausgaben erfassen, damit meine Kosten nachvollziehbar sind.
5. Als Benutzer moechte ich eine Finanzuebersicht sehen, damit ich Einnahmen, Ausgaben und Saldo erkenne.
6. Als Benutzer moechte ich Monatsbudgets pro Ausgabenkategorie erfassen, damit ich Ausgaben kontrollieren kann.
7. Als Benutzer moechte ich sehen, ob ein Budget ueberschritten wurde.
8. Als Benutzer moechte ich meine Daten in einer Datenbank speichern, damit sie nach dem Neustart erhalten bleiben.

## Optionale Erweiterungen

- Filter nach Konto, Kategorie und Monat
- Diagramme zur Visualisierung
- Monatsvergleich
- CSV-Export
- Login fuer mehrere Benutzer
